Hello World：我的第一个网站诞生记

旅程的开始

欢迎来到我的博客！ 这是我发布的第一篇文章，标志着我的个人网站 luckyou.asia 终于正式上线了。从一个想法到拥有一个可以随时访问的网站，这个过程充满了学习和挑战，也带来了巨大的成就感。

整个过程就像是探索一片新大陆，每解决一个问题，地图上就点亮一个新的区域。

我都做了些什么？

能成功看到这个页面，说明我已经完成了以下所有步骤：

  1. 购买域名：拥有了 luckyou.asia 这个独一无二的地址。
  2. 购买服务器：在阿里云拥有了一台ECS云服务器作为网站的“家”。
  3. 域名解析：设置了DNS，让域名能够准确找到服务器的IP地址。
  4. 部署Web服务器：在服务器上安装并运行了强大的 Nginx。
  5. 配置安全证书：通过 Certbot 申请了免费的SSL证书，让网站实现了 https:// 安全访问。
  6. 学习 Hugo：选择了 Hugo 这个静态网站生成器来构建我的博客内容。

未来的计划

这个网站目前还很简单，但它是一个全新的起点。接下来我计划：

  • 分享更多关于技术的学习笔记。
  • 记录生活中有趣的事情。
  • 把网站的样式打造得更具个性化。

如果你对我建站的过程感兴趣，可以看看下面这个 Nginx 的重启命令，这是我经常用到的一个指令。

# 重启 Nginx 服务的命令
sudo systemctl restart nginx
